# Netflix Frontend Clone

A simple **Netflix landing page and login page clone** built using **HTML and CSS only**.  
This was my **first frontend project**, created to learn webpage structuring, styling, and responsive layout techniques.  
The UI is designed to look very close to the original Netflix website.

---

## Features

- Pixel-perfect **Netflix landing page clone**
- **Login page UI** similar to the official site
- Built completely from scratch using **HTML + CSS**
- Beginner-friendly folder and code structure

---

## Tech Stack

- **HTML5**  
- **CSS3**

---

## Learning Outcomes

- Understanding **layouts** (Flexbox, Grid)
- Working with **background images, overlays, gradients**
- Improving skills in **UI cloning and attention to detail**

## Future Improvements

- Add smooth animations and transitions
- Improve mobile responsiveness
- Add JavaScript for interactive elements
- Create more pages (Browse page, Footer links, etc.)

1. Clone the repository
2. Open the folder
